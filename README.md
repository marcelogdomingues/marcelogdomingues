### 👨‍💻 About me

- 🔭 Currently building a backend for remotely configuring EV chargers over **OCPP 2.0.1**
- 🌱 Digging into **BPMN orchestration**, validation architecture and async deployment lifecycles
- 🧱 Java / Spring on the server, Angular on the client, Postgres & MySQL underneath
- 📍 Based in **Lisbon**
- 💬 Ask me about **Spring Boot, Angular, SQL dialects & data migrations**
